Tecnologias utilizadas:
- Radix UI - Para acessibilidade;
- Json server - Para criação de API;
- Biblioteca Axios - Para fazer as requisições http.
- EsLint - Para organização do código.

Funcionalidades:
- Busca;
- Adição de nova transação.
